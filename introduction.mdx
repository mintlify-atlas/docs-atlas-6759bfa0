    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get NearYou running on your device in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the NearYou repository to your local machine:
      
      ```bash
      git clone https://github.com/edgarconrado/nearyou.git
      cd nearyou
      ```
    </Step>

    <Step title="Install dependencies">
      Install the required dependencies using your preferred package manager:
      
      <CodeGroup>
        ```bash npm
        npm install
        ```

        ```bash yarn
        yarn install
        ```

        ```bash pnpm
        pnpm install
        ```
      </CodeGroup>
    </Step>

    <Step title="Start the development server">
      Launch the Expo development server:
      
      ```bash
      npx expo start
      ```
      
      <Note>
        This will open Expo Developer Tools in your browser. From there, you can run the app on an emulator or physical device using the Expo Go app.
      </Note>
    </Step>

    <Step title="Explore on your device">
      Open the Expo Go app on your iOS or Android device and scan the QR code to launch NearYou. Start discovering places near you!
    </Step>
  </Steps>
</div>
